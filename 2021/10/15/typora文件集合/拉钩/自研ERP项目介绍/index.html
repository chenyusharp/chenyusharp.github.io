个人自我介绍

你好，我叫夏振宇，现在是谊品生鲜商品模块的owner。我们商品中台是谊品生鲜的基础数据团队，负责销售商品和供应链商品的创建以及维护，对公司的各个业务线提供商品的相关服务。团队现在有7个人，每个人都负责一个相应的模块。自己主要负责把控整个商品的架构、模型设计，同时推进核心项目的落地以及一些业务需求的迭代。

项目经历

一、自研ERP项目

项目的起因&项目的背景？

公司目前有两套ERP系统和一套自研中台，用以支持谊品生鲜和到家业务，其中ERP承载了：

  1. 供应链（采购、库存）业务；
  2. 总部经营管理（供应商、商品类、机构类基础数据管理和经营报表）
  3. 门店销售及经营报表
  4. 财务业务（富集）

但是，2套ERP存在下面的问题：

  1. 2套商品主数据、2套模型，通过中台商品映射erp商品编码完成；但是缺少有效的校验，导致毛利错误、仓库作业困难；
  2. 两套ERP未对谊品生鲜、到家业务进行隔离，无法相互兼容，影响业务深度展开；
  3. 业务需求响应慢，无法及时满足业务需求；
  4. 深度定制费用高；

涉及的工程：ypshop项目（门店端）、ypsx-matrix、ypsx-titan、

涉及的表：sg_base_item

达成了什么目标？

重新设计了谊品自己的商品模型，其中包括基础资料（主档），完成了主档的创建、修改以及分档创建商家商品、地点商品，另外增加了对于生鲜商品的支持：PLU码的管理、价签模版管理、BOM物料管理、生鲜品调价、商品生命周期管理等功能。
通过这些功能，可以实现自主商品建档、分档，商品售价/进价管理、商品生命周期管理、支持多形态商品销售（BOM）等，打通了主档、货品、销售商品的创建、维护，可以有效的支持谊品自己的进销存结（进货、销售、库存、结算）核心业务；

遇到了哪些困难？怎么克服的？最终取得了什么样的结果？
  1. 为了更好的支持商品的创建，我们对部分商品的表进行了分库分表的设计，根据门店id和商品的编码；
  2. 各个业务的查询逻辑复杂，不能做到和模型相对应，对此，增加了多个ES索引，用以支持不同场景的查询；
  3. 针对上传、下载的场景比较多，比较消耗系统资源，开发出了独立的人task模块；
  4. 针对分库分表主键id需要全局唯一的问题，引入了分布式id的生成；（具体参考分布式id的生成）；
自己承担了什么职责？

核心能力技术方案的输出与开发，包括分档、调价、BOM管理、ES等模块；协调商品接口与外域的联调等。对技术难点进行攻关与测试，以及项目的开发排期。
收获：

  1. 对谊品的业务模式，尤其是生鲜中涉及的业务场景有了更加深入的了解；
  2. 增加了系统设计的能力，认识到技术方案的设计需要结合业务场景来看（可以举下调价单中sg_price_modify_docs这个索引设计中遇到的坑）；
  3. 大型项目中团队协作开发经验，系统之间的联调、沟通能力的都增强；

二、谊批宝项目

职业规划

  1. 希望往业务专家方向发展，同时增进技术深度，拓宽业务视野；
  2. 阿里的平台够大、基础设施搭建的好、技术深厚，同时有丰富的业务场景，有利于在开发道路上走的更长远；
  3. 希望自己的商品业务经验能给公司带来绵薄之力，起到双赢的效果；

为何跳槽

  1. 个人家庭原因（社保问题）；
  2. 平台限制发展，业务范围有限，缺少更大的发展空间；
  3. 大流量高并发场景不足；

个人成长以及优缺点

个人成长

  1. 开发思维有所改变，按照DDD的领域模型思维进行思考问题；
  2. 有带团队经验，现在团队成员6人；
  3. 业务深度提高，跨域协调、项目管理能力得到增强，（采购、wms、营销、导购、到家等）；
  4. 用技术解决业务问题的能力得到了进一步的提高；（ES同步、导入导出等）；

优缺点

优点：

  1. 做事情有恒心，有着强烈的责任心；
  2. 热衷于学习新技术；
  3. 现在有一定的供应链商品相关的行业经验；

缺点：

  1. 缺少大厂履历，项目比较局限于自营业务；
  2. 事必躬亲，有时候会影响到团队其他成员的成长以及项目的进度；

工作经历的亮点：

  1. 独立开发并重构了商品数据同步到ES的项目，使商品同步到ES的时效性得到大幅度的提升；
  2. 领导商品从阿里云到腾讯云的基础设施迁移工作，涉及多达10几个系统的迁移（DB、Redis、ES等），并使业务平滑过度；
  3. 有丰富的项目实施经验，参与了商品历次大型的项目的实施；
  4. 发现业务使用产品过程中的痛点，并提供相关的技术改进措施；（task任务、es搜索、数据下发）

软性能力的评估：

  1. 坚持学习，自驱性强；
  2. 多思考、多实践；

性格特点：

坚毅、乐观、好学；

做事风格：

  1. 做任何事情之前，习惯列好计划；
  2. 好记性不如烂笔头，多做笔记；
  3. 坚持、持之以恒，尽自己最大的努力完成；
